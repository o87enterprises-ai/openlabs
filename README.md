# OpenLabs: Virtual Chem Lab (Beta)
A real-world physics, live 3D/VR research lab.

## Mission
We are removing all barriers to knowledge. OpenLabs is an initiative to provide high-fidelity, accessible scientific environments for everyone, everywhere. 

> "The only asset is time. The only currency is love, and the only debt is fear." - OpenEdin

## Proof of Concept
This is a beta prototype demonstrating a Vinegar + Baking Soda reaction in a WebGL-powered 3D environment.
